# FlintOS-Arduino
A ultra-lightweight, monolithic GUI micro-OS designed for the Arduino Uno (ATmega328P) and MCUFRIEND 3.5" TFT displays. Features custom boot diagnostics, a master PIN passcode lock screen, a dark/light theme engine, a touch-canvas Notepad app, and 3 integrated game engines (Racing, Snake, Flappy Bird) engineered to run entirely within 2KB of SRAM.
